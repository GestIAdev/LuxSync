## üé® DESIGN PRINCIPLES

### Visual Hierarchy:
1. **Ember Theme Consistency**: Using CSS custom properties ensures coherence with the rest of the app
2. **Subtle Guidance**: UX hints at 40% opacity - visible but not intrusive
3. **Interactive Feedback**: Hover glow and slide animation confirm user action
4. **Depth Perception**: Triple-layer shadows and backdrop blur create floating effect

### Typography:
- **Hints Text**: System font for readability, weight 500 for presence without boldness
- **Letter Spacing**: 0.5px for better legibility at small size (10px)

### Color Philosophy:
- **Primary Accent**: Orange (#ff6b2b) - energy, warmth, action
- **Dark Surface**: #1c1c1e - deep but not pure black
- **Glows**: 30% opacity for ambient, 15% for tints - subtle but noticeable
