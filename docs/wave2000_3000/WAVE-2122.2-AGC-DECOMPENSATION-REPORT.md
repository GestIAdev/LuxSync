## 🔬 ROOT CAUSE (La Mentira del rawBassEnergy)

### El Flujo Real de la Señal

```
ringBuffer (4096 samples, audio REAL)
  ↓
AGC.processBuffer(buffer) ← MODIFICA EL BUFFER IN-PLACE
  │                           gain varía 0.59x - 0.82x frame a frame
  ↓
buffer ya está COMPRIMIDO por AGC
  ↓
GodEarAnalyzer.analyze(buffer) → FFT sobre señal COMPRIMIDA
  ↓
rawBands = energía por banda (calculada del FFT post-AGC)
  ↓
rawBassEnergy = rawBands.subBass + rawBands.bass
  │
  └──→ ESTA ENERGÍA HEREDA LA MODULACIÓN DEL AGC
       El nombre "raw" es UNA MENTIRA
       Es "raw" respecto al band-AGC (capa 2)
       Pero es POST buffer-AGC (capa 1)
```

### Por Qué 95 BPM y No 126 BPM

El AGC tiene un smoothing window de **15 frames × 46.4ms = ~700ms**.

Esto crea un ciclo de "pumping" del gain que:
1. Sube cuando la energía baja (inter-beat)
2. Baja cuando la energía sube (kick)
3. El ciclo completo pump-release ≈ 630ms → **~95 BPM**

La autocorrelación del `rawBassEnergy` ve:
- **Lag 13 (≈95-100 BPM):** correlación alta → CICLO DE PUMPING DEL AGC
- **Lag 10 (≈126 BPM):** correlación débil → beat real ENMASCARADO

El harmonic sieve no puede ayudar porque el peak a L13 NO tiene sub-armónicos — es un artefacto puro, no una relación armónica real.

## 🔧 LA CORRECCIÓN

### Principio Matemático

```
Energía ∝ Amplitud²
AGC multiplicó la amplitud por gain
→ E_post = E_real × gain²
→ E_real = E_post / gain²
```

### Código (senses.ts)

```typescript
// WAVE 2122.2: AGC DECOMPENSATION
const agcGain = agcResult.gainFactor > 0.01 ? agcResult.gainFactor : 1.0;
const trackerEnergy = spectrum.rawBassEnergy / (agcGain * agcGain);
```

## 🏗️ Las Dos Capas de AGC

```
CAPA 1: Buffer AGC (AutomaticGainControl.ts)
├── Modifica waveform in-place
├── Gain: 0.5x - 8.0x
├── Smoothing: 15 frames (~700ms)
├── ⚠️ CONTAMINA rawBassEnergy
└── FIX: dividir por gain²

CAPA 2: Band AGC (GodEarFFT.ts → BandAGC)
├── Normaliza por banda después del FFT
├── rawBands BYPASA esta capa ✓
└── bands (con AGC) usa esta capa
```

**WAVE 2122.2 corrige la contaminación de Capa 1. Capa 2 nunca fue el problema.**
