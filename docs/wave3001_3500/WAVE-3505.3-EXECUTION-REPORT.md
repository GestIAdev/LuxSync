
#### Static Curve Functions

| Function | Curve | Use Case | Formula |
|----------|-------|----------|---------|
| `applyLinear(v)` | 1:1 | Generic fade | `v` |
| `applyExponential(v, exp=2.5)` | Power law | Percussion snappy | `v ^ exp` |

## SYSTEM IMPLEMENTATIONS

### ImpactSystem.ts — Audio-Reactive Lighting Physics

**Lines:** 346 | **NodeFamily:** `IMPACT` | **Owns:** `dimmer`, `shutter` (L0)

#### Concept
Translates percussive audio energy (kicks, drops, envelope transients) into intensity commands for impact nodes. Respects role-specific band mixes and envelope decay rates.

#### Band Mix Matrix (7 bands × 4 roles)

```
Role        | subBass | bass  | mid  | highMid | presence | air  | energy
------------|---------|-------|------|---------|----------|------|-------
percussion  |  0.80   | 0.60  | 0.10 |  0.05   |  0.00    | 0.00 | 0.20
breath      |  0.10   | 0.20  | 0.70 |  0.15   |  0.10    | 0.05 | 0.30
accent      |  0.00   | 0.05  | 0.10 |  0.70   |  0.80    | 0.60 | 0.20
ambient     |  0.10   | 0.15  | 0.10 |  0.10   |  0.10    | 0.35 | 0.30
```

**Singletons:** All matrices are `Object.freeze()`-d — zero-alloc reads in hot path.

#### Envelope Decay Model

```
ATTACK:     Instantaneous (new audio value jumps immediately)
            → Kick feels snappy, responsive

DECAY:      Role-specific exponential fall
  percussion: 300ms  (fast decay, quiet gap between hits)
  breath:    1200ms  (organic, breathing persists)
  accent:    150ms   (staccato, sharp strobes)
  ambient:   3000ms  (lingering presence)
```

#### Floor (Minimum Intensity)

```typescript
// ambient/breath never collapse to absolute zero in silence
const floor = {
  breath:  0.05,    // 5% minimum presence
  ambient: 0.08,    // 8% ambient lingers
  primary: 0.03,    // 3% for primary
  others:  0,       // percussion/accent hit zero
}
```

#### Integration Example

```typescript
// ImpactSystem.process() hot path
view.forEach((node: IImpactNodeData) => {
  // 1. Select band mix for this role
  const weights = this._resolveBandMix(node.role, vibe)

  // 2. Compute raw energy (dot product of 7 bands)
  const rawEnergy = BaseSystem.computeBandMix(audio, weights)

  // 3. Apply transfer curve (exponential for percussion = snappy)
  const shapedValue = this._applyNodeCurve(rawEnergy, node)

  // 4. Envelope decay (in-place mutation of node.envelopeState)
  const envelopeValue = this._stepEnvelope(node, shapedValue, deltaMs)

  // 5. Apply floor (ambient doesn't die)
  const dimmer = this._applyFloor(envelopeValue, node.role)

  // 6. Apply vibe master intensity
  const finalDimmer = BaseSystem.clamp01(dimmer * vibe.intensity)

  // 7. Write to scratch (zero-alloc)
  this._intentScratch.nodeId = node.nodeId
  this._valuesDict['dimmer'] = finalDimmer

  // Conditional strobe for accents + transient
  if (node.role === 'accent' && audio.hasTransient && audio.transientStrength > 0.7) {
    this._valuesDict['shutter'] = 1.0
  } else if ('shutter' in this._valuesDict) {
    delete this._valuesDict['shutter']
  }

  // 8. Push to bus (O(1), bus copies values)
  bus.push(this._intentScratch as INodeIntent)
})
```

### ColorSystem.ts — Multi-Type Chromatic Management

**Lines:** 415 | **NodeFamily:** `COLOR` | **Owns:** `red`, `green`, `blue`, `white`, `cyan`, `magenta`, `yellow`, `color_wheel` (L0)

#### Mixing Types Supported

| Type | Output Channels | Use Case | Conversion |
|------|-----------------|----------|-----------|
| `rgb` | R, G, B | Generic LED | Direct from palette |
| `rgbw` | R, G, B, W | CRI-efficient LEDs | Extract white component |
| `cmy` | C, M, Y | Subtractive filters | `C=1-R, M=1-G, Y=1-B` |
| `wheel` | color_wheel (DMX) | Mover color wheel | Nearest-neighbor slot |
| `hybrid` | Wheel + RGB(w) | Smart fixtures | Saturated→wheel, pastel→RGB |

#### Role-Based Color Assignment

```typescript
// What color does each node receive?
role: 'primary'    → palette[0]  (dominant color)
role: 'accent'     → palette[1]  (accent color)
role: 'pixel'      → gradient along palette (deterministic per index)
role: 'decoration' → blend palette[0] + palette[1] by harmonic tension
role: 'others'     → palette[0] default
```

#### LERP Temporal (Anti-Flicker)

```typescript
// Smooth color transitions per frame
const lerpSpeed = lerp(
  COLOR_LERP_SPEED_BASE,    // 250ms settle time (normal)
  COLOR_LERP_SPEED_MAX,     // 80ms settle time (high energy)
  audio.energy              // scales by audio energy
)
const lerpT = clamp01(lerpSpeed * deltaMs)

// Evaluate LERP
node.currentColor.r = lerp(node.currentColor.r, targetRgb.r, lerpT)
node.currentColor.g = lerp(node.currentColor.g, targetRgb.g, lerpT)
node.currentColor.b = lerp(node.currentColor.b, targetRgb.b, lerpT)
```

#### White Extraction (RGBW Efficiency)

```typescript
// For RGBW fixtures, extract common component as white LED
const wComponent = min(r, g, b)
if (wComponent > WHITE_EXTRACTION_THRESHOLD) {
  r_out = r - wComponent
  g_out = g - wComponent
  b_out = b - wComponent
  w_out = wComponent  // ← white LED covers all three
}
```

**Benefit:** Better color rendering index (CRI) + lower power consumption.

#### Color Wheel Nearest-Neighbor

```typescript
// For mechanical wheels: find closest slot by Euclidean distance in RGB space
let bestDmx = slots[0].dmxValue
let bestDistSq = Infinity

for (const slot of slots) {
  const sr = slot.previewRgb.r / 255
  const sg = slot.previewRgb.g / 255
  const sb = slot.previewRgb.b / 255
  const dr = target.r - sr, dg = target.g - sg, db = target.b - sb
  const distSq = dr*dr + dg*dg + db*db

  if (distSq < bestDistSq) {
    bestDistSq = distSq
    bestDmx = slot.dmxValue
    if (distSq < THRESHOLD) break  // close enough
  }
}

return bestDmx / 255  // normalize to [0, 1]
```

### KineticSystem.ts — VMM Deterministic Movement

**Lines:** 298 | **NodeFamily:** `KINETIC` | **Owns:** `pan`, `tilt`, `pan_fine`, `tilt_fine`, `rotation`, `speed` (L0)

#### VMM Patterns (All Deterministic, No Math.random())

| Pattern | Trigger | Description | Phase Offset |
|---------|---------|-------------|--------------|
| `idle` | Base state | Static home position (0.5, 0.5) | None |
| `sweep` | Verse/ambient | Sinusoidal pan + tilt sync'd to BPM | `stereoIndex` (snake wave) |
| `scatter` | Break | Fixed dispersed positions per node | `stereoIndex / total` |
| `converge` | Bridge | All nodes point same (tensioned) point | Tension-modulated tilt |
| `wave` | Transitional | Traveling wave across array | `(stereoIndex + beatPhase) % 1` |
| `build` | Pre-drop buildup | Tilt rises progressively | Progress× intensity |
| `drop` | Drop moment | Dramatic impact position + settle | Stutter-free animation |

#### Stereo Routing

```typescript
// Nodes on left side (physical position x < 0) mirror pan from right
if (node.physicalPosition.x < 0) {
  pan = 1 - pan  // ← creates symmetric fan-open/fan-close effect
}
```

#### Gearbox Budget (Motor Speed Limiting)

```typescript
// Respect hardware constraints: don't exceed max pan/tilt speed
// maxPanSpeed in deg/s → normalize to units/ms
const maxPanDeltaPerMs = node.maxPanSpeed / (540 * 1000)
const maxTiltDeltaPerMs = node.maxTiltSpeed / (270 * 1000)

// Compute desired movement
const panDelta = target_pan - current_pan
const tiltDelta = target_tilt - current_tilt

// Clamp to motor budget
const limitedPan = currentPan + clampDelta(panDelta, maxPanDeltaPerMs * deltaMs)
const limitedTilt = currentTilt + clampDelta(tiltDelta, maxTiltDeltaPerMs * deltaMs)

// Update node position in-place
node.currentPosition.pan = limitedPan
node.currentPosition.tilt = limitedTilt
```

#### Fine Channels (Slow Motors)

```typescript
// For stepper motors (slow, maxPanSpeed < 90 deg/s), use fine resolution
if (node.maxPanSpeed < SLOW_MOTOR_THRESHOLD) {
  const panFraction = limitedPan * 255
  this._valuesDict['pan_fine'] = panFraction % 1  // fractional part
}
```
